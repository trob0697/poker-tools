# PokerToolsV2

A website containing tools useful to pokers players.  
Tech Stack:
- React.js
- Express.js
- PostgreSQL
- Knex.js
- Docker

## Getting Started
1. Clone repository
```
$ git clone https://github.com/trob0697/pokertools-v2.git
```
2. Open local instance
```
$ cd pokertools-v2
```
3. Bring up
```
$ docker compose up app
```
