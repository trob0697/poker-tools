# Poker Tools
## Getting Started
1. Clone repository
```
$ git clone https://github.com/trob0697/poker-tools.git
```
2. Open local instance
```
$ cd poker-tools
```
3. With docker installed, bring up containers
```
$ docker compose up -d front-end-dev back-end-dev
```
4. Access app at http://localhost:3000
